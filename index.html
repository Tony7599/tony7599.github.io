Network Blocker Utility

This tool allows you to temporarily block network access for any running application.

Important: You'll need administrator privileges to run this batch file.

How to use:

  1. Download the batch file below
  2. Right-click the file and select "Run as administrator"
  3. Select the application you want to block from the list
  4. Keep the window open to maintain the block
  5. Close the window to restore network access

Note: This will download a .bat file that creates temporary Windows Firewall rules.
