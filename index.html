躲避老师appVeyon监控

此工具允許您暫時阻止任何正在執行的應用程式的網路訪問。

Important: 您需要管理員許可權才能執行此批處理檔案。

使用:

  1. 下載下面的批處理檔案
  2. 右鍵單擊檔案並選擇「以管理員身份執行」
  3. 從列表中選擇您要阻止的應用程式
  4. 保持視窗開啟以保持塊
  5. 關閉視窗以恢復網路訪問
  6. 找到[xxx]server-veyon.exe 在输入框输入[xxx]的数字

Note: This will download a .bat file that creates temporary Windows Firewall rules.
